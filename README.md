# Let’s write a theorem prover!

Some material I am playing with about implementing a theorem prover. Very rough ideas are in [/docs](docs). The `W_n` branches are for different versions of the VM of increasing complexity in `n`. 

The theorem prover is realized by implementing a Prolog-like language using Warren’s Abstract Machine.
